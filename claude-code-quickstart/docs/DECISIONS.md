# Claude Code Quick Start - Decisions

## D001 - Scope

Built this as a multi-page follow-along guide rather than a reference dump. The user explicitly wanted a comprehensive beginner path that teaches fundamentals while getting the learner to a working Claude Code setup.

## D002 - Visual Family

Kept the typography, dark/light mode, and sidebar language close to `zero-to-hero-programming`, but simplified the interaction model and page density so it feels like a quick-start sibling, not a second full curriculum.

## D003 - Technology Baseline

Used static HTML, CSS, and vanilla JS only, matching the existing repository style and avoiding any build system.

## D004 - Claude Code Facts

Used the native installer as the recommended install path and explicitly marked npm installation as deprecated because official Anthropic docs checked on April 6, 2026 describe the native installer as the preferred path.

## D005 - Beginner Safety

Did not teach destructive Git recovery commands. The guide emphasizes practice repos, `git status`, `git diff`, small approvals, and frequent commits instead.

## D006 - Orchestration Framing

Included a high-level orchestration chapter because the user asked for the bigger mental model, but explicitly separated "for now" from "for later" so the learner does not confuse advanced concepts with day-one tasks.
