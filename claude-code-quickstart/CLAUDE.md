# Claude Code Quick Start Site

## Purpose
- Build a beginner-first static site that takes a complete novice from zero terminal knowledge to a safe first Claude Code session.
- Keep the material sequential, practical, and follow-along.

## Pedagogy Rules
- Define every technical term before using it.
- Explain every important command in plain English and show how to verify success.
- Teach the safe path first: practice repo, plan mode, small edits, frequent commits.
- Separate "use now" from "learn later" whenever advanced orchestration appears.
- Use absolute dates for volatile Claude Code facts.

## Content Rules
- Official Anthropic docs outrank all other sources for Claude Code behavior and install guidance.
- Do not recommend destructive Git recovery commands to beginners.
- Do not imply that free web chat access automatically grants Claude Code access.
- Keep the site visually aligned with the existing learning materials.

## Build Rules
- Static HTML, CSS, and vanilla JS only.
- No build step.
- Shared styles live in `css/design-system.css`.
- Shared scripts live in `js/theme.js`, `js/sidebar.js`, and `js/shared.js`.
